# DeepCable Audit — 电缆清单标准化审计工具

解析电缆招标/报价 Excel 清单，按 **GB/T 19666** 标准化型号，校验 N/PE 截面，聚合数量，输出 6-Sheet 专业采购审计报告。

## 工作流

```
原始清单.xlsx
      │
      ▼
  ┌─────────────────────────────┐
  │ read_cable_list()           │  ← 自动检测型号列、数量列、项目特征列
  │ 保留非电缆行（非电缆标记）     │     多 Sheet 合并，非米单位标记
  └─────────────────────────────┘
      │
      ▼
  ┌─────────────────────────────┐
  │ 预处理                       │  ← 2*(YJV...) 并联翻倍
  │                              │     裸芯规格（3×25+2×16）加前缀
  └─────────────────────────────┘
      │
      ▼
  ┌─────────────────────────────┐
  │ normalize()                 │  ← 型号标准化引擎
  │ GB/T 19666 前缀规范化        │     材质互斥修正(YJV→YJY)
  │ 电压等级解析与补全            │     旧标→新标映射(NHVV→N-YJV)
  │ 耐火/阻燃等级标准化           │     非标英文写法映射
  └─────────────────────────────┘
      │
      ▼
  ┌─────────────────────────────┐
  │ check_pe()                  │  ← N/PE 截面校验
  │ 按 GB/T 16895 规则检查       │     中性线、保护接地线截面是否合理
  └─────────────────────────────┘
      │
      ▼
  ┌─────────────────────────────┐
  │ aggregate()                 │  ← 按型号指纹聚合数量
  │ 单芯线拆分(BV-2×2.5→)       │     中压→电力→控制→布电线→软线→弱电排序
  └─────────────────────────────┘
      │
      ▼
  ┌─────────────────────────────┐
  │ build_output()              │  ← 6-Sheet Excel 输出
  │ 原始清单(原样保留) /          │     电缆提取+标准化 / 聚合采购清单 /
  │ 型号解释 / 待处理项 / 操作日志  │
  └─────────────────────────────┘
      │
      ▼
  <项目名>_电缆审计_<日期>.xlsx
```

## 快速开始

```bash
# 一键处理
python deepcable_run.py 原始清单.xlsx

# 指定项目名（影响输出文件名）
python deepcable_run.py 原始清单.xlsx --project 某某产业园

# 裸芯规格自动加前缀
python deepcable_run.py 原始清单.xlsx --prefix WDZB-YJY

# 指定输出目录
python deepcable_run.py 原始清单.xlsx --out ./reports/
```

## 输出文件（6-Sheet Excel）

| Sheet | 内容 | 特点 |
|-------|------|------|
| **原始清单** | 客户文件原样复制 | 保留原始格式（合并单元格、字体、颜色、边框），冻结保护 |
| **电缆提取+标准化** | 原始 vs 标准化左右对比 | 颜色编码：橙=修正、绿=无修、灰=已排除，PE 校验，超链接追溯 |
| **聚合采购清单** | 对外交付采购表 | 按品类分组，数量验算式，来源超链接 |
| **型号解释** | 通俗中文说明 | 采购语言去术语化，含适用场景和注意事项 |
| **待处理项** | 人工介入清单 | 失败/警告/待确认/已修正四类，留白填写处理结果 |
| **操作日志** | 审计追溯 | 时间戳 + 处理阶段 + 操作描述 + 规则依据 |

## 标准化引擎特性

覆盖 **GB/T 19666** 全部标准前缀组合及主流制造商企标：

- **阻燃等级**：ZA/ZB/ZC/ZD/ZB1/ZB2 — 含 WD（无卤低烟）与含卤体系
- **耐火等级**：N/NH/NS/NA — 自动修正 NH→N
- **绝缘材料**：YJY（XLPE）/ YJV（老标含卤）/ PVC / 聚烯烃 — 材质互斥检测（YJV→YJY）
- **护套/铠装**：22（钢带）/ 32（钢丝）/ 62（非磁性）/ V / Y / R（软）
- **电压等级**：0.6/1kV / 8.7/10kV / 26/35kV 自动解析与补全
- **矿物电缆**：BBTRZ / RTTZ / BTLY 等柔性防火电缆
- **控缆/布电线**：KYJY / KYJYP / BV / BVR / BYJ / BYJR / RVS / RVSP 等
- **特殊型号**：防水 FS-、带形 -D-、阻水 ZS- 等

### 非标映射

支持多种非标准/英文写法自动识别：

`NHYJV-0.6/1KV-5X16` → `N-YJV-0.6/1kV-5×16`

`CU/XLPE/PVC/SC` → `YJV`（根据结构拆解）

`RS485 2×1.5` → `ZC-RVSP-300/300V-2×1.5`

## 项目结构

```
deepcable-audit/
├── deepcable_run.py          # CLI 入口 — 一键运行
├── deepcable_read_excel.py   # Excel 读取 & 自动列检测
├── deepcable_normalize.py    # 型号标准化（对外接口）
├── normalizer.py             # 标准化引擎核心
├── tokenizer.py              # 型号分词器
├── parser.py                 # 型号解析器
├── spec.py                   # 规格数据模型
├── aggregator.py             # 数量聚合 & 排序
├── validation.py             # N/PE 截面校验
├── model_interpreter.py      # 型号→中文解释
├── excel_output.py           # 6-Sheet Excel 生成
├── tests/
│   └── batch_normalize.py    # 批量测试
├── references/               # 标准 & 制造商企标
│   ├── gbt19666_prefix.md    # GB/T 19666 前缀组合规则
│   ├── pe_table.md           # PE 截面最小要求
│   ├── mapping.md            # 非标/英文/旧标写法映射
│   ├── rules.md              # 复杂边界规则
│   ├── voltage.md            # 电压等级锁定表
│   └── 万马股份_*/           # 各制造商企标（6家40+文件）
└── SKILL.md                  # Claude Code skill 定义
```

## 参考标准

- **GB/T 19666** — 阻燃和耐火电线电缆通则
- **GB/T 12706** — 额定电压 1kV 到 35kV 挤包绝缘电力电缆
- **GB/T 16895** — 建筑物电气装置（PE/N 截面规则）
- **JB/T 8734** — 额定电压 450/750V 及以下聚氯乙烯绝缘电缆电线和软线
- **制造商企标**：万马股份、上上电缆、上海起帆、宝胜创新、成天泰（成天泰）、远东电缆
