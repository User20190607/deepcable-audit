# GB/T 19666-2019 燃烧特性代号体系

## 来源
GB/T 19666—2019 阻燃和耐火电线电缆或光缆通则（代替 GB/T 19666—2005）

---

## 1. 全部代号（表1）

| 代号 | 名称 | 说明 |
|------|------|------|
| **Z** | 单根阻燃 | 含卤产品 Z 可省略 |
| **ZA** | 阻燃A类 | 成束，7L/m，40min |
| **ZB** | 阻燃B类 | 成束，3.5L/m，40min |
| **ZC** | 阻燃C类 | 成束，1.5L/m，20min（常规默认） |
| **ZD** | 阻燃D类 | 成束，0.5L/m，20min（d≤12mm或S≤35mm²） |
| **W** | 无卤 | pH≥4.3, 电导率≤10μS/mm, HCl+HBr≤0.5%, HF≤0.1% |
| **D** | 低烟 | 最小透光率≥60% |
| **U** | 低毒 | 毒性指数≤5（推荐值） |
| **N** | 耐火 | 90min供火+15min冷却，2A熔断器不断 |
| **NJ** | 耐火+机械冲击 | 120min，IEC 60331-1/2 |
| **NS** | 耐火+冲击+喷水 | 120min，最后15min水喷淋/喷射 |

---

## 2. 阻燃系列组合（表2）

### 含卤阻燃
`Z` `ZA` `ZB` `ZC` `ZD`

### 无卤低烟阻燃
`WDZ` `WDZA` `WDZB` `WDZC` `WDZD`

### 无卤低烟低毒阻燃
`WDUZ` `WDUZA` `WDUZB` `WDUZC` `WDUZD`

---

## 3. 耐火系列组合（表3）

耐火级别 N / NJ / NS 叠加到任一阻燃前缀之后。

| 系列 | 示例 | 含义 |
|------|------|------|
| 含卤 N | N, ZAN, ZANJ, ZANS | 阻燃A类耐火(NS) |
| 无卤低烟 N | WDZN, WDZAN, WDZCNJ, WDZANS | C类NS耐火 |
| 无卤低烟低毒 N | WDUZN, WDUZANJ | 低毒A类NJ耐火 |

> **关键约束（5.2节）：耐火产品必须同时具备阻燃性能。**
> 即 N/NJ/NS 不能独立存在，前缀中必须有 Z。

---

## 4. 前缀结构规则

```
[W]─[D]─[U]─Z[A~D]─[B1/B2]─[N/NJ/NS]
  │    │    │    │       │        │
  │    │    │    │       │        └─ 耐火类型 (可选, 必须搭配Z)
  │    │    │    │       └─ 燃烧等级 GB 31247 (可选)
  │    │    │    └─ 阻燃等级 (必选，Z=单根)
  │    │    └─ 低毒 (可选, 必须搭配WD)
  │    └─ 低烟 (必须搭配W)
  └─ 无卤 (可选)
```

| 约束 | 规则 |
|------|------|
| D 前必须有 W | 不存在 `DZ*` |
| U 前必须有 W+D | 不存在 `WU*`（缺低烟）或 `UZ*`（缺无卤低烟） |
| N/NJ/NS 前必须有 Z | 耐火必须同时阻燃 (5.2) |
| B1/B2 | 来自 GB 31247，不是 GB/T 19666 代号，但在产品型号中可标注 |

## 5. 标准化写法示例

```
示例1: ZB-YJV-0.6/1 4×120
       含卤B类阻燃电力电缆

示例2: WDZANS-YJY-0.6/1 3×70+1×35
       无卤低烟A类NS耐火电力电缆

示例3: WDUZCNJ-KYJY-450/750 7×2.5
       无卤低烟低毒C类NJ耐火控制电缆

示例4: WDZ-BYJ-450/750 1×2.5
       无卤低烟单根阻燃布电线（非C类！）
```

## 6. 常见错误及修正

| 错误 | 正确 | 原因 |
|------|------|------|
| WDZ → WDZC | **WDZ 保留** | WDZ=单根阻燃，合法型号 |
| ZN-YJV | ZAN-YJV 或 N-YJV | 含卤阻燃+耐火必须带等级字母，单根耐火直接用 N |
| NH | N | GB/T 19666 废弃 NH |
| ZR / ZRC | ZC | 非标写法 |
| N-ZC | ZCN | 顺序修正 |
| DWZB | WDZB | 逆序修正 |
| WDZDN → WDZCN | **不应自动转换** | WDZDN=无卤低烟D类阻燃耐火，合法 |
| WDZD → WDZC | **不应自动转换** | WDZD=单根D类阻燃，适用于小电缆 |

## 7. 阻燃等级选择指南

| 场景 | 推荐等级 |
|------|----------|
| 一般建筑配电 | ZC（C类，常规默认） |
| 重要公共建筑 | ZB（B类） |
| 超高层/地铁/核电站 | ZA（A类） |
| 小直径电缆（d≤12mm） | ZD（D类） |
| 单根敷设、无成束风险 | Z（单根） |
