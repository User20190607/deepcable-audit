## 软导体无卤布电线对应关系

| 含卤型号 | 无卤对应型号 | 说明 |
|---|---|---|
| BV | BYJ | 硬导体，无护套 |
| BVR | BYJR | 软导体，无护套，禁止误映射为BYJ |
| RV | RYJ | 软导体，无护套（连接软线级） |
| BVV | BYJY | 硬导体，有护套 |
| RVV | RYJY / RYY | 软导体，有护套 |
| RVVP | RYJYP | 屏蔽软护套线 |
| RVSP | RYJSP | 屏蔽双绞线 |
| RVS | RYJS | 双绞线 |

> ⚠ R（软导体）≠ 无R（硬导体），严禁互换。RYJ 无护套，RYJY 有护套，不同型号。
