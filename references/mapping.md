| RV | RYJ | 软导体，无护套（连接软线级） |

## GB/T 19666-2019 燃烧特性前缀矩阵

### 阻燃系列（表2）

| 阻燃等级 | 含卤 | 无卤低烟 | 无卤低烟低毒 |
|----------|------|----------|-------------|
| 单根 | Z | WDZ | WDUZ |
| A 类 | ZA | WDZA | WDUZA |
| B 类 | ZB | WDZB | WDUZB |
| C 类 | ZC | WDZC | WDUZC |
| D 类 | ZD | WDZD | WDUZD |

> Z(单根)仅通过单根垂直燃烧。ZA~ZD 为成束阻燃，试验梯次严苛。

### 耐火后缀（表3，叠加到任一阻燃前缀后）

| 后缀 | 含义 | 供火时间 | 附加条件 |
|------|------|----------|----------|
| N | 单纯供火耐火 | 90min供火+15min冷却 | 2A熔断器不断 |
| NJ | 供火+机械冲击 | 120min | GB/T 19216 / IEC 60331 |
| NS | 供火+冲击+喷水 | 120min | 最后15min喷淋 |

> ⚠ GB/T 19666 5.2：耐火产品**必须同时具备阻燃性能**。N/NJ/NS 不能独立存在，必须有 Z 前缀。

### 前缀组合规则

```
W → D → [U] → Z[A~D] → [B1/B2] → [N/NJ/NS]
```

| 规则 | 说明 |
|------|------|
| D 前必须有 W | 不存在 DZ-、DZA- |
| U 前必须有 W+D | 不存在 WU-（缺低烟）, UZ- |
| N/NJ/NS 前必须有 Z | 耐火必须同时阻燃 |
| Z 不带字母 = 单根 | WDZN ≠ WDZCN |
| B1/B2 来自 GB 31247 | 燃烧等级，非 19666 代号体系 |

### 常见错误前缀修正

| 输入 | 修正 | 原因 |
|------|------|------|
| NH / NH- | N | GB/T 19666 废弃 NH |
| ZR / ZRC | ZC | 非标写法 |
| N-ZC / NH-ZC | ZCN | 顺序修正 |
| DWZ* | WDZ* | 逆序修正 |
| WDNH- | WDZN- | 废弃代号合并 |

### 前缀合法性快速校验

| 校验项 | 非法示例 | 正确 |
|--------|----------|------|
| 含卤阻燃+耐火必须带等级 | ZN-YJV | ZAN-YJV / ZBN-YJV / ZCN-YJV（单根耐火直接用 N） |
| 低毒必须有WD | UZC-BYJ | WDUZC-BYJ |
| 低烟必须有W | DZB-YJV | WDZB-YJV |
| D类仅小电缆 | WDZD-YJY-3×400 | 截面≤35mm² 或 外径≤12mm |

|---|---|---|
