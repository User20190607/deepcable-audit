# 全套处理规则 v24.2

## §1 材质互斥防火墙

**触发条件**：前缀中含有 `WD`（含 WDZA/WDZB/WDZC/WDZN/WDZCN/WDUZ 及与N/NS/NJ组合变体），且型号部分含 `V`（PVC含卤材质），强制修正。

N/NS/NJ 单独作前缀时（如 `N-KVV`）不触发。

| 原始型号 | 修正后型号 |
|---|---|
| WDZN-YJV | WDZN-YJY |
| WDZN-VV | WDZN-YJY |
| WDZN-KVV | WDZN-KYJY |
| WDZN-RVV | WDZN-RYJY |
| WDZCN-RVVP | WDZCN-RYJYP |
| WDZCN-RVSP | WDZCN-RYJSP |
| WDZ-RVS / WDZN-RVS | WDZ-RYJS / WDZN-RYJS |
| WDZN-DJYPV | WDZN-DJYYP |
| WDUZC-YJV | WDUZC-YJY |
| WDZC-BVR / WDZN-BVR | WDZC-BYJR / WDZN-BYJR |
| WDZC-RV / WDZN-RV | WDZC-RYJ / WDZN-RYJ |

> ⚠ WDZ- 和 WDZN- 是完整合法前缀，修正仅针对 V 材质，不改前缀本身。WDU 系列无豁免。

---

## §2 B1 燃烧性能等级处理

-B1- 标识必须原样保留，不得删除或移位。

| 情形 | 执行动作 |
|---|---|
| WD前缀 + Y材质 + -B1- | 合规，保持原样 |
| WD前缀 + V材质 + -B1- | 修正 V→Y，-B1- 保留 |
| 无WD前缀 + -B1-（如 ZCN-RVS-B1） | 补全WD前缀，修正V→Y，-B1- 保留 |
| 无 -B1- 后缀 | 不主动升级无卤，保持原样 |

---

## §3 前缀标准化

### §3.1 标准转换表

| 原始前缀 | 修正后前缀 | 说明 | 适用范围 |
|---|---|---|---|
| NH | N | 仅保留N，不添加阻燃等级 | 全系列 |
| ZDN- | ZCN- | 非法前缀，直接修正 | 全系列 |
| WDZDN- | WDZCN- | 非法前缀，直接修正 | 全系列 |
| WDZD-（非标） | WDZC- | 非法前缀，直接修正 | 全系列 |
| ZR（无等级）/ ZRC | ZC | 旧写法C级阻燃 | 仅低压（≤0.6/1kV） |
| ZRA | ZA | A级阻燃，禁止降级 | 仅低压 |
| ZRB | ZB | B级阻燃，禁止降级 | 仅低压 |
| ZRD | ZD | D级阻燃 | 仅低压 |
| 中压（>0.6/1kV） | 保持原样 | 中压不执行ZR→Z转换 | 中压以上 |
| ZC-BYJ / ZCN-BYJ | WDZC-BYJ / WDZCN-BYJ | 普通阻燃+无卤材质→补WD | 无卤项目 |
| WDZ- / WDZN- | 保持原样，禁止补全为WDZC-/WDZCN- | 完整合法表述 | 全系列 |

> ⚠ ZRA/ZRB 绝对禁止降级为 ZC，属重大工程安全事故。

### §3.2 反向材质冲突（普通前缀+无卤材质）

触发：核心材质为纯无卤型（BYJ/YJY/KYJY/RYJY），但前缀不含WD。

- 已确认项目无卤标准 → 自动补WD前缀
- 新项目意图不明 → 暂停询问

### §3.3 基础线材保护

`BV / BVR / RV / RVS / RVV / RVVP`（无WD前缀版本）禁止主动添加阻燃或无卤前缀，仅补电压。

### §3.4 用户指定型号豁免（最高优先级）

用户明确确认过的非标前缀（如WDU-）和特殊型号（RYS/RYJ/RV），字面忠诚，严禁自动修正。V材质冲突仍须修正。

---

## §4 特殊型号规则

### §4.1 RVS 芯数限制
RVS 仅2芯，出现3芯及以上时暂停询问，禁止自动修正。

### §4.2 RYJ / BYJR 软导体无卤布电线

| 型号 | 对应含卤型号 | 禁止误映射为 |
|---|---|---|
| RYJ（铜芯XLPE绝缘软电线，无护套） | RV | BYJ（硬导体） |
| RYJY（铜芯XLPE绝缘聚烯烃护套软电线） | RVV | RYJS / BYJ |
| BYJR（铜芯XLPE绝缘软导体布电线） | BVR | BYJ（硬导体） |

### §4.3 光伏电缆
`PV1-F` → `PV1-F-DC1500V-1×（截面）`

### §4.4 特殊工艺标识保留
`(F)`（辐照交联）、`(G)`（固定敷设）、`22/23/32/33`（铠装代号）、`-B1-`（燃烧等级）原样保留。

### §4.5 中压电缆豁免（≥6kV）
前缀保持原样，不执行ZR→Z转换，不执行材质互斥修正。

---

## §5 格式标准化

### §5.1 符号与单位

| 项目 | 规则 |
|---|---|
| 乘号 | * → × |
| 单位 | 统一使用 m（禁止中文米） |
| 截面单位 | 去除 mm2 / mm² 后缀 |
| 单芯补全 | 缺芯数时补全 1× |

### §5.2 括号清理
`1(4×35+1×16)` → `4×35+1×16`

### §5.3 无前缀型号处理
已知项目标准→按标准补前缀；新项目/前缀不明→暂停询问。禁止无前缀直接输出。

### §5.4 YJY截面下限保护
截面 < 1.5mm² 时暂停询问：转为×1.5 还是改用KVV系列？

### §5.5 异常标识检测
NI/NII/NIII（桥架耐火等级，JB/T 10216）出现在电缆型号中时，暂停询问是否为录入错误。

---

## §6 数量计算

### §6.1 单位换算
- 卷/扎/盘（截面≤10mm²）：1卷=100m
- 卷/扎/盘（截面≥16mm²）：按清单标注实际长度；仅有卷数则标注[需核实实际长度]

### §6.2 数量向上取整
有小数一律向上进位（只多不少）。

### §6.3 结构矛盾拆分

| 原始写法 | 处理方式 |
|---|---|
| BV 2×2.5 | 拆为 BV-1×2.5，数量×2 |
| 2*(3×6)+1×4（星三角） | 拆为 3×6 一根 + 3×6+1×4 一根 |
| 3X(4XBVR150)+BVR240 | BVR-1×150（×12根）+ BVR-1×240（×1根） |

### §6.4 多图累加
默认全量累加，不自动去重。用户明确说明重复时才去重。

---

## §7 单芯线颜色处理

- 颜色顺序：黄→绿→红→蓝→黑→白→双
- 单芯线格式：`黄1绿1红3蓝5双5`（颜色+卷数，紧连无空格，无卷字，无等号）
- 双绞线格式：成对出现，`红蓝3500 红黄500`（不拆分为红1750蓝1750）

---

## §8 型号指纹严格匹配

合并前必须验证五项全部一致：前缀 + 核心型号 + B1后缀 + 电压等级 + 截面规格。任一不同即为独立物料。
